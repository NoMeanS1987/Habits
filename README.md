# 🌱 habits

Персональный трекер привычек с RPG-механикой. Сделан для себя, постепенно оброс фичами и превратился в полноценное мини-приложение.

Работает как Telegram Mini App — открывается прямо внутри Telegram через кнопку меню бота.

→ [Открыть приложение](https://nomeans1987.github.io/Habits)

---

## Что умеет

**Привычки**
- Отмечать выполнение за каждый день недели
- Несколько режимов повтора: каждый день, по будням, по выходным, свои дни, каждые N дней
- Цветовая маркировка для каждой привычки
- Персональные напоминания в выбранное время (в твоей таймзоне)
- Серия дней подряд и общая статистика

**Задачи**
- Разовые и повторяющиеся задачи
- Напоминания в конкретное время
- Фильтры: сегодня / неделя / все

**Тренировки**
- Журнал тренировок с упражнениями, весом, подходами и повторами
- Каталог из 27 упражнений по группам мышц + свои упражнения
- Таймер отдыха между подходами

**Помодоро**
- Круговой таймер, режимы 25/5 и 45/15
- Счётчик сессий за день

**RPG-система**
- Четыре характеристики: 💪 Сила, 🧠 Интеллект, ⚡ Ловкость, ❤️ Здоровье
- XP и уровни за выполнение привычек и задач
- Сложность задачи влияет на награду
- Карточка героя со статистикой

---

## Стек

**Фронтенд** — чистый HTML/CSS/JS без фреймворков. Один файл `index.html`.
Деплой: GitHub Pages

**Бэкенд** — FastAPI + PostgreSQL + aiogram 3
- Синхронизация состояния между устройствами через REST API
- Telegram-бот: утренний и вечерний дайджест, персональные напоминания
- Авторизация через Telegram WebApp initData (HMAC-SHA256)
- Деплой: Railway

→ [habits-backend](https://github.com/NoMeanS1987/habits-backend)
